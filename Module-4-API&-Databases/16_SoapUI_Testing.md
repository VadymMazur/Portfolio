# 🧼 SOAP API Testing

Please open the file **"CountryInfoService-soapui-project"** to review the comprehensive test suites and service mockups.

---

### 📂 Getting Started:
1. Launch **SoapUI** (Open Source or ReadyAPI).
2. Go to `File` -> `Import Project`.
3. Select the `CountryInfoService-soapui-project.xml` file.
4. Navigate through the **TestSteps** to examine the assertions and XML requests.

### 🔍 What's Inside:
* Functional test cases for country data retrieval.
* Validation of SOAP responses against WSDL definitions.
* Automated test scripts for service reliability.
